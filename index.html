Visual Studio Code is an offline IDE meaning you can use it even without wifi or internet. You can just set it up by downloading it of chrome.

The console screen is used to print stuff from js to the html page. To open it you have to right click on the page after clicking "live preview" on the code screen. Then, click console and view it all. If you want to print anything on the console you just type your code then put console.log(your variable/storage thing).
